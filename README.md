Task Manager API (Java)

Projeto simples de backend desenvolvido em Java para praticar conceitos fundamentais de desenvolvimento de APIs.

Funcionalidades

Criar uma tarefa

Listar todas as tarefas

Buscar tarefa por ID

Atualizar tarefa

Deletar tarefa

Estrutura do projeto

O projeto segue uma arquitetura em camadas:

Controller → recebe as requisições
Service → contém a lógica de negócio
Repository → gerencia os dados
Model → estrutura da entidade Task

Tecnologias utilizadas

Java

IntelliJ IDEA

Programação Orientada a Objetos

HashMap como armazenamento em memória

Objetivo

Este projeto foi desenvolvido como prática de backend antes de evoluir para uma versão utilizando Spring Boot e banco de dados.
